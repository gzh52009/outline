# 项目说明

基于Vue框架的webApp项目

## 项目类型
* 类型不限，建议非电商类型

## 项目要求
* 完善github仓库资料
* 爬取数据到本地数据库
* 编写数据API
* 移动端适配
* 编写项目说明文档README.md并放到项目根目录，文件需包含以下内容
    * 项目名称
    * 演示
        * 官网
        * 上线网址
    * git仓库地址
    * 项目页面截图
        > 截有代表性得页面，不少于3张
    * 项目目录说明


## 技术栈

### 前端
* Vue
* Vue-Router
* Vuex
* Vue-Cli
* axios
* elementUI/MintUI/iView/antd/vantUI
* git

### 后端
* Nodejs
* Express
* MongoDB（推荐）/mySQL


## 项目包含模块
不同类型的项目有不同的模块，不做硬性要求，可以参考自己选择的项目，但必须包含以下模块
* 注册
* 登录

## 项目完成后所需操作
* 备份数据库
    > 存入项目`db`目录
* 上线
