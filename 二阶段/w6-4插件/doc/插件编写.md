### 扩展内置对象方法

* Number
* String
* Boolean
* Object

    - Function
    - RegExp

### 自定义插件

## 了解jQuery的原理
通过扩展jQuery原型对象（`jQuery.prototype`）来增加本身的功能

**【案例】**

* 日期格式化方法
* 弹出层插件
* ajax插件
    - get()
    - post()
    - jsonp()
* 简化版jquery
    > 面向对象的DOM操作
    - show()：显示
    - hide()：隐藏
    - on()：事件绑定
    - animate()：动画
    - css()：获取/设置Css样式
    - addClass()：添加类名
    - removeClass()：移除类名
    - attr():获取/设置html属性值
    - html()：设置/获取html内容
